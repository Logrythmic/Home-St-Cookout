# Home-St-Cookout

## Description
This application will allow two sets of users, vendors & consumers, to participate in or host pop-up restaurant/food events in their local areas.

The app will contain the following features:
- For Vendors:
- - Event creation
- - Menu updates
- - Order managment

- For Consumers:
- - Local event board
- - RSVP
- - Pre-Order
- - Calender Reminders
